# Multimedia

Esta carpeta contiene todas las imágenes e iconos del proyecto DondeOficial.

## Estructura

### Imágenes
- **`imagenes/globales/`** - Imágenes globales del sitio (logos, banners, etc.)
- **`imagenes/negocios/`** - Imágenes de los negocios registrados
- **`imagenes/categorias/`** - Imágenes representativas de cada categoría

### Iconos
- **`iconos/globales/`** - Iconos globales del sitio
- **`iconos/negocios/`** - Iconos de los negocios
- **`iconos/categorias/`** - Iconos de las categorías

## Uso

Para usar las imágenes en los componentes, importa desde esta carpeta:

```typescript
import imagenEjemplo from '@/multimedia/imagenes/globales/logo.png';
```

## Formatos recomendados

- **Imágenes**: PNG, JPG, WebP
- **Iconos**: SVG, PNG (con transparencia)
